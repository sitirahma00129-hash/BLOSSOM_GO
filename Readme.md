# 🌸 Blossom GO
### Official Korean Album & Merchandise Store

Blossom GO adalah website e-commerce yang menyediakan layanan pembelian Korean Album dan Merchandise K-Pop secara online. Website ini dibuat dengan desain modern, responsif, dan memiliki fitur interaktif untuk memberikan pengalaman belanja yang mudah bagi pengguna.

---

## 📌 Project Overview

Blossom GO merupakan website toko online yang berfokus pada penjualan:

- 💿 Korean Album
- 💡 Official Lightstick
- 🛍️ K-Pop Merchandise
- ⭐ Best Seller Products

Website ini dirancang dengan tampilan aesthetic menggunakan konsep warna lembut yang sesuai dengan identitas brand Blossom GO.
